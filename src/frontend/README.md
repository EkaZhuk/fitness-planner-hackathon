# Frontend (Streamlit)

## Файлы
| Файл | Назначение |
|:---|:---|
| `app.py` | Главный файл приложения |
| `components.py` | Переиспользуемые UI-компоненты |
| `utils.py` | Вспомогательные функции |

## Страницы приложения
1. **Цель** — установка целевой дистанции и даты забега
2. **Добавление тренировки** — форма ввода данных
3. **Прогресс** — график дистанции по неделям и ML-прогноз

## Запуск
```bash
streamlit run src/frontend/app.py
