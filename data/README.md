# Данные проекта

## Источник
Синтетические данные, сгенерированные с помощью библиотеки Faker (Python).

## Формат данных
Файл `training_data.csv`

| Поле | Тип | Описание |
|:---|:---|:---|
| date | YYYY-MM-DD | Дата тренировки |
| distance_km | float | Пройденная дистанция в километрах |
| duration_min | float | Длительность в минутах |
| avg_pace | float | Средний темп (мин/км) |
| notes | string | Заметки пользователя |

## Объём
~100 записей (имитация 3-4 месяцев тренировок)

## Генерация
Скрипт генерации находится в `/notebooks/01_generate_data.ipynb`
