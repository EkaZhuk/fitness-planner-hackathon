# fitness-planner-hackathon
Планировщик тренировок с ML-прогнозированием прогресса

**Команда ____**

## Описание проекта
Приложение для планирования тренировок с функцией ML-прогнозирования прогресса.

**Цель:** Помочь бегунам-любителям отслеживать прогресс и прогнозировать достижение целевой дистанции.

**ML-составляющая:** Линейная регрессия для предсказания финишной дистанции на заданную дату.

## Стек
- Backend: FastAPI
- Frontend: Streamlit
- ML: Scikit-learn
- Данные: Pandas + Faker (синтетика)

## Структура
- `/data` — сгенерированные данные тренировок
- `/notebooks` — Jupyter ноутбуки с анализом
- `/src/frontend` — Streamlit приложение
- `/src/backend` — FastAPI эндпоинты
- `/src/ml` — обучение и инференс модели
- `/docs` — документация и прототипы
